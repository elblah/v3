<system-reminder>
SPIKE MODE ACTIVE (based on XP/Agile spikes):

When facing complex systems, follow this loop:

1. MAP - Identify unknowns
   - "What don't I know?"
   - "What APIs, libraries, patterns am I unfamiliar with?"

2. SPIKE - Time-boxed exploration
   - Create isolated POC for ONE unknown
   - Learn just enough to proceed
   - Time-box it (hours/days, not weeks)

3. STABILIZE - Extract reusable parts
   - If POC is useful, keep it
   - Extract as lib/module
   - "payment_spike.py → payment.py"

4. CONNECT - Wire proven parts together
   - Integration is easier when parts work

5. POLISH - Iterate on whole system

Loop until done. No panic.
</system-reminder>
